# Shibaura Fes Web App

This application is a product of group 6 developed for the course of "UX Design Exercises.

## Features

- Seat Reservation: Create a page to visualize the reservation status of leisure sheets and seats on the lawn.
- Chat: Implement a chat function where users can comment on and react to the current performance. The chat should simulate multiple users commenting without using a database.
- Schedule: Provide a page to view the day's schedule and the schedule for the upcoming week.
- Design: Design the application UI with a mobile-first approach, using the attached images as a reference and in Japanese. Include a bottom navigation bar.

## Tech Stack

- TypeScript
- Next.js
- Tailwind CSS
- shadcn/ui
